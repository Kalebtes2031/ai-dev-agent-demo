# ai_dev_agent

Clean scaffold for the ai_dev_agent project.

## Setup

1. Create and activate a virtual environment.
2. Install dependencies from `requirements.txt`.

## Run

```powershell
python main.py
```

## Structure

- `bot/` - bot-related modules (placeholder)
- `agent/` - agent logic (placeholder)
- `executor/` - execution layer (placeholder)
- `data/` - local data assets (placeholder)
- `projects/` - project workspaces (placeholder)
- `main.py` - entry point
